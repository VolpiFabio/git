------ FLUXO DE TRABALHO -------

1. criar o diretório (pasta) do projeto
2. iniciar o git dentro do diretório criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add (nome arquivo)
6. git status
7. git commit -m 'mensagem'
8. git status
9. volta para o passo 3.

git init . = inicia o repositorio na pasta atual
git status = mostra o status atual do diretorio
git add nome_do_arquivo = prepara o arquivo para commit
git commit -m 'primeiros comandos bash' = descrição do commit
git config --global user.email 'email do usuario'
gir config --global user.name 'nome do usuario'
git logs = mostra o histórico de commits
git add . = adiciona tudo da pasta atual
git checkout nome_da brach = troca a branch atual
git branch = mostra quantas branchs tem criada
git checkout -b nome_da_branch = cria uma nova branch
git branch -d nome_da_branch = deleta a branch
git diff nome arquivo = retorna o que foi adicionado ou retirado do arquivo
git reset nome arquivo = retira um arquivo de staged (que estaria pronto para comitar)
